HTML links or hyperlinks are HTML elements that allow us to redirect website users from one webpage to another webpage.

HTML links or hyperlinks can be created with < a > < /a > (anchor) tags.

The href attribute represents the destination page where we want to be redirected.

The target attribute represents how we want to be redirected on a new page.

_self
redirects on the same page tab
_blank
redirects on a new page tab

Text link is an HTML link that is displayed like a plain text inside of a webpage.

Image link is an HTML link that is displayed like an image text inside of a webpage.
