cellpadding defines the distance between cell content and cell border, while cellspacing defines the distance between table cells.

colspan is an HTML attribute that allows us to merge two or more columns in one, while rowspan is an HTML attribute that allows us to merge two or more rows in one.

Table sections are predefined positions in a table. All rows that belong to the < thead > < /thead > section will be displayed at the top of the table. In addition, all rows that belong to the < tbody > < /tbody > section will be displayed in the middle of the table. Moreover, all rows that belong to the < tfoot > < /tfoot > section will be displayed at the bottom of the table regardless of whether we place that section on the top or on the bottom of the < table > < /table > HTML element.
