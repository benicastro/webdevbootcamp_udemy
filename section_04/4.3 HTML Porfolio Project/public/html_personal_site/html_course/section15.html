Section 15: HTML Forms

What I've learned:

  1. What is an HTML form ?
  2. HTML form structure
  3. Important form attributes

HTML form is an HTML element that we use to get data from website users and send that data to a server for server-side data processing.

What are the important form attributes?

  • action
  • method
  • target

The action attribute is where we set the name of the server-side script that will do the data processing.

The method attribute is where we set the HTTP method that we will use to send our data on the server for data processing.

The target attribute is where we want to display the result of server-side data processing.
