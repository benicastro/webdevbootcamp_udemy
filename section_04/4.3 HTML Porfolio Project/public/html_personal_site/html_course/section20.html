Section 20: Head Element

What I've learned:

  1. What is head element?
  2. What is base element?
  3. What is link element?
  4. What is meta element?
  5. What is script element?
  6. What is style element?
  7. What is title element?

The head element is used to define styles, scripts, and meta data related to the HTML document.

The base HTML tag is used to define the base URL that will be used for all relative URLs in the HTML document.

The link HTML tag is used to link the current HTML document with some external source.

The meta HTML tag is used to define various meta data for the HTML document.

The script HTML tag is used to define various scripts for the HTML document.

The style HTML tag is used to define CSS styles for the HTML document.

The title HTML tag is used to define the title for the HTML document which is visible inside the tab of the web browser.
