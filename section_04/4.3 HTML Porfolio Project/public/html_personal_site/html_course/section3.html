HTML Attributes usually contain the name of the attribute and the value of the attribute.

The most common HTML attributes are:

  • id - allows us to set unique identifier to an HTML element
  • title - allows us to give title to an HTML element, usually displayed as tooltip text for that element
  • class - is used to add styles from CSS file to HTML element
  • style - is used to add inline styles that are available in CSS (colors, margins, paddings, etc.) to HTML element without CSS classes
