Section 21: Meta Tags

What I've learned:

  1. What are meta tags?
  2. What is the stricture of meta tags?
  3. What is the author meta tag?
  4. What is the description meta tag?
  5. What is the keywords meta tag?
  6. What is the revised meta tag?
  7. What is the refresh meta tag?
  8. What is the content-type meta tag?
  9. What is the viewport meta tag?

The meta tag is used to define various instructions and information that are important for the HTML document.

The author meta tag is used to set who is the author of the website.

The description meta tag is used to set the description of the website.

The keywords meta tag is used to set keywords that are website related.

The revised meta tag is used to set the date of the last update of the website.

The refresh meta tag is used to set a period of time for the website to be refreshed.

The content-type meta tag is used to define what is the type of document we send.

The viewport meta tag is used to define how the content of the website will be scaled on small size devices.
