Section 9: HTML Lists

What I've learned:

  1. What are lists ?
  2. What are unordered lists ?
  3. What are ordered lists ?
  4. What are description lists ?

Lists are HTML elements that allow us to handle arrays of data in a way that is best for us. There are three types:

  1. unordered lists < ul > < /ul >
  2. ordered lists < ol > < /ol >
  3. description lists < dl > < /dl >

1. Unordered lists allow us to handle data in an unordered way.

  • Element 3
  • Element 1
  • Element 2
  • Element 4
  • Element 3
  • Element 1
  • Element 2
  • Element 4
  • Element 3
  • Element 1
  • Element 2
  • Element 4

2. Ordered lists allow us to handle array of data in an ordered way. We can order our data numerically or alphabetically.

Default-Case Numerals (Type 1):

  1. Element 1
  2. Element 2
  3. Element 3
  4. Element 4

Upper-Case Numerals (Type I):

  1. Element 1
  2. Element 2
  3. Element 3
  4. Element 4

Lower-Case Numerals (Type i):

  1. Element 1
  2. Element 2
  3. Element 3
  4. Element 4

Upper-Case Letters (Type i):

  1. Element 1
  2. Element 2
  3. Element 3
  4. Element 4

Lower-Case Letters (Type i):

  1. Element 1
  2. Element 2
  3. Element 3
  4. Element 4

Description lists are used in cases where we have data that contains an element and the description of that element.
