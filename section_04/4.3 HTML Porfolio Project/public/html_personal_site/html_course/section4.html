Section 4: HTML Paragraph and Text Formatting

What I've learned:

  1. What is an HTML paragraph or <p> </p>?
  2. What is <br> </br> HTML element?
  3. What is <pre> </pre> HTML element?
  4. Text Formats:
    • bold or <b> </b>
    • italic or <i> < /i >
    • underlined or <u> </u>
    • strikethrough or <strike> </strike>
    • monospaced or <tt> </tt>
    • superscript or <sup> </sup>
    • subscript or <sub> </sub>
    • bigger or <big> </big>
    • smaller or <small> </small>
    • marked or <mark> </mark>

An HTML Paragraph element allows us to create text paragraphs inside of our webpage.

A Line Break HTML element is an element that we use inside of our text where we want to start a new line for further text.

A pre-formatted HTML element is an element that follows all additional spaces between words and all new lines that we define in our original text.
