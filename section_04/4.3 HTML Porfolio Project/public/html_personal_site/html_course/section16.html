HTML form elements are HTML elements that we use to use to create HTML forms.

What are the common form elements?

  • input
  • select
  • textarea
  • button
  • datalist

The < input > element is an HTML element used to create various types of form elements like text field, password field, submit button, reset button, radio buttons, checkboxes, etc.

The < select > element is an HTML element used to create a drop-down list.

The < textarea > element is an HTML element used to create a multiline input text field.

The < button > element is an HTML element used to create clickable buttons.

Types of buttons:

button
button for which we need to define an on-click function
submit
button that takes all values from HTML form and sends it to the server for data processing
reset
button that represents all values from HTML form on initial values

The < datalist > element is an HTML element that represents combined versions of input text field and drop-down list.

The list attribute is where we set the data list that we want to connect with the input field. This value needs to be the same as the id value for data list.
