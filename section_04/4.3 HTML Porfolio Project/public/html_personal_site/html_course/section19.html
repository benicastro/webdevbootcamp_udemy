Section 19: Restriction Attributes

  6. What is step attribute?
  7. What is pattern attribute?
  8. What is min and max attributes?
  9. What is minlength and maxlength attributes?

Restriction attributes are attributes that we use to restrict the default behavior of HTML elements.

The disabled attribute is a restriction attribute that we use to disable an HTML element.

The readonly attribute is a restriction attribute that we use to restrict an HTML element where its value can only be read and cannot be changed.

The required attribute is a restriction attribute that we use to require web users to enter values in the specific field.

The size attribute is a restriction attribute that we use to define or restrict the width of an input field measured by the number of characters.

The step attribute is a restriction attribute that we use to define a valid step that will be used in the input field.

The pattern attribute is a restriction attribute that we use to define a pattern that will be needed to populate the input field.

The min and max attributes are restriction attributes that we use to define the minimum and maximum values needed to populate the input field.

The minlength and maxlength attributes are restriction attributes that we use to define the minimum and maximum number of characters needed to populate the input field.
