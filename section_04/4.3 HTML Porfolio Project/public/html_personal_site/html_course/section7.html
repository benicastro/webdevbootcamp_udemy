Section 7: HTML Comments

What I've learned:

  1. What are comments ?
  2. Why to use comments?
  3. What are comment types ?
  4. What are valid and invalid comments?

Comments are HTML elements that we write inside of our code and these elements are not visible inside of our webpage.

<!-- This is a comment -->

Comments are very useful for creating reminders inside the code for yourself or for other people.

Comments are also very useful for debugging code in cases when our code doesn't work as it should.

There are two types of comments:

  • one line
  • multiline
