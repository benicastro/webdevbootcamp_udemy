Section 2: HTML Document Structure

What I've learned:

  1. <!DOCTYPE html>
  2. <html> </html>
  3. <head> </head>
  4. <body> </body>
  5. Parent and Child relationship

The !DOCTYPE element indicates what version of HTML do we use.

The html element contains the head and body elements.

The head element contains styles, scripts, and instructions.

The body element represents the visible part of the webpage.

A Parent element is an element that contains another element while a Child element is an element that is nested inside of another element.
