Section 11: HTML Images

What I've learned:

  1. What is an image HTML element ?
  2. How to set image source location?
  3. How to set image height and width?
  4. How to set image border?
  5. How to set image alignment?

The Image < img /> HTML element is used to insert images inside of webpages.

The src attribute represents the name of the image file that we want to display.

The alt attribute represents the text that will be displayed in case the image cannot be loaded for some reason.

The image height and width can be set with the HTML attributes height and width.

The image border can be set with the border HTML attribute that we add inside the image HTML element.

Image alignment can be set with the align HTML attribute that we set inside the HTML image element.
