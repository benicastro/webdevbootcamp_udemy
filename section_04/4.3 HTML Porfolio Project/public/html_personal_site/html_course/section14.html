Section 14: HTML Iframe

What I've learned:

  1. What is HTML Iframe ?
  2. What are Iframe attributes ?

HTML Iframe is an HTML element that allows us to create a rectangular region inside of our HTML document where we can load the content of another HTML document or the content of another website.


< iframe src="IframeContent" >< /iframe >


What are Iframe attributes?

  • src
  • frameborder
  • width
  • height
  • marginwidth
  • marginheight
  • scrolling

src is used to set the HTML document or website address that we want to load like the content of the iframe.

frameborder is used to enable or disable the frame border that goes around of the iframe.

width is used to set the width for the iframe.

height is used to set the height for the iframe.

marginwidth represents the distance between the left and right frame border, and frame content.

marginheight represents the distance between the top and bottom frame border, and frame content.

scrolling is used to enable or disable scrolling inside of the iframe.

The important attributes for the frameset element include cols and rows that represent the number of columns or rows of the screen division, respectively, while for the frame element, we have src that represents the document that will be loaded, and name which represents the unique name for that frame.
