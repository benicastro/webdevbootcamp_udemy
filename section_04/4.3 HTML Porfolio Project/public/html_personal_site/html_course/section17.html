The < input > element is an HTML element that we use to create various type of elements like:

  • input text field
  • password field
  • radio buttons
  • checkboxes

The basic structure of an input element is:

<input type="inputType" name="inputName" value="initialValue"/>

type
represents the type of input element
name
represents the unique name for the input element
value
represents the initial value for the input element

The different input element types are:

  • text
  • password
  • reset
  • submit
  • radio
  • checkbox
  • button
  • file
  • image
